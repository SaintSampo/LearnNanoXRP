💻 ¡Bienvenido al Editor Web de XRP!

Ahora que tu robot está construido, ¡es hora de hacer que haga cosas increíbles! Escribirás programas usando una herramienta llamada el Editor Web de XRP.

El Editor Web de XRP — ¡tu base de operaciones para programar!
🌐
Abre el editor

Tu maestro te dará la dirección web. Escríbela en tu navegador Chrome o Edge y presiona Enter. ¡El editor funciona directamente en tu navegador web, sin necesidad de instalación!

En esta lección aprenderás:

  • Cómo conectar tu robot al editor
  • Cómo navegar por el editor
  • Cómo escribir tu primer programa en Blockly
  • Cómo ejecutar tu código en el robot

🔌 Conectando Tu Robot

Antes de programar, necesitas conectar tu robot a la computadora con un cable USB.

Enchufa el cable USB en el robot y en tu computadora
1

Conecta el cable USB

Conecta un extremo del cable USB al puerto en la placa controladora del robot, y el otro extremo a tu computadora.

2

Enciende el robot

Mueve el interruptor de encendido en la placa controladora a la posición ON. Deberías ver una luz LED verde encenderse.

3

Haz clic en "Connect" en el editor

En el editor web, haz clic en el botón Connect en la parte superior. Aparecerá una lista de puertos. Elige uno que diga algo como "XRP" o "COM" y haz clic en Connect.

4

Busca el indicador verde

Cuando esté conectado, el editor muestra un indicador verde "Connected". ¡Estás listo para programar!

🗺️ Tour del Editor

Veamos rápidamente las partes principales del Editor Web de XRP.

Las áreas principales del Editor Web de XRP

Áreas Clave:

A

Barra de Herramientas (arriba)

Botones para Conectar, Ejecutar, Detener y Guardar tu programa.

B

Caja de Herramientas de Bloques (izquierda)

Categorías de bloques que puedes usar. Haz clic en una categoría para ver los bloques dentro.

C

Espacio de Trabajo de Blockly (centro)

El área grande donde arrastras y conectas bloques para escribir tu programa.

D

Vista de Código Python (lado derecho)

Esto te muestra el código Python que generan tus bloques. ¡Muy genial!

E

Terminal (abajo)

Muestra mensajes de tu robot — como si algo salió mal, o si usaste un bloque de impresión.

🧱 ¿Qué es Blockly?

Blockly es una forma de escribir programas encajando bloques — ¡como piezas de un rompecabezas! Cada bloque hace una cosa específica.

Los bloques se encajan para construir un programa — ¡sin necesidad de escribir!

✅ Blockly

Arrastra y suelta bloques coloridos. Es difícil cometer errores de tipeo. ¡Excelente para principiantes!

🐍 Python

Escribe el código tú mismo. Más poderoso, pero tienes que conseguir que cada letra sea perfecta.

🤔
¿Sabías que?

Cuando usas Blockly, el editor automáticamente convierte tus bloques en código Python real. Los ingenieros profesionales usan Python para programar robots, satélites y mucho más.

Los bloques se organizan en categorías por color. Aquí están los más importantes para el NanoXRP:

Haz clic en cualquier categoría para ver los bloques dentro
🚗

DriveTrain (naranja óxido)

Hace que el robot avance, retroceda y gire.

📡

Sensores (verde)

Lee el sensor de distancia, sensores de línea, giroscopio y más.

💡

Placa de Control (azul verdoso)

🎮

⚙️

Control (azul grisáceo)

Bucles, sentencias if, bloques de espera — la lógica de tu programa.

✋ Tu Primer Programa

¡Hagamos que el robot parpadee su luz LED! Este es el clásico "Hola Mundo" de la robótica.

¡Tu primer programa: haz parpadear el LED encendido y apagado!

Construye este programa paso a paso:

1

Encuentra el bucle "forever"

En la categoría Control, encuentra el bloque repeat forever y arrástralo al espacio de trabajo.

2

Añade bloque LED On

En Control Board, arrastra LED on dentro del bucle forever.

3

Añade un bloque de espera

En Control, encuentra wait [0.5] seconds y añádelo después de LED on.

4

Añade LED Off y otra espera

Añade LED off y otro bloque wait 0.5 seconds.

⏱ wait 0.5 seconds

💡 LED off

⏱ wait 0.5 seconds

▶️ Ejecutando Tu Código

Una vez que tus bloques están en su lugar, ¡es hora de enviar el programa al robot y verlo ejecutarse!

Haz clic en el botón verde Run para enviar tu programa al robot
1

Asegúrate de que el robot está conectado

Verifica que el indicador verde "Connected" esté mostrándose en la barra de herramientas.

2

Haz clic en el botón Run ▶️

El editor convierte tus bloques a Python y lo envía al robot. La Terminal mostrará qué está pasando.

3

¡Observa tu robot!

El LED en la placa debe parpadear encendido y apagado cada medio segundo. ¡Felicidades — acabas de ejecutar tu primer programa de robot!

4

Detén el programa

Haz clic en el botón rojo Stop ⏹ para detener la ejecución del programa. El LED dejará de parpadear.

⚠️
¿Algo salió mal?

Verifica el área de Terminal en busca de mensajes de error en rojo. El problema más común es no estar conectado — intenta hacer clic en Connect de nuevo.

💾 Guardando Tu Trabajo

¡Guardar tus programas es importante! Aquí está cómo guardar y volver a abrir tu trabajo.

Guarda tu programa para que puedas volver a él más tarde
1

Guarda al Robot

Haz clic en File → Save to Robot. Tu programa se guarda en el robot mismo como un archivo llamado main.py. ¡Este programa se ejecuta automáticamente cuando el robot se enciende!

2

Guarda en la Computadora

Haz clic en File → Save para guardar una copia en tu computadora. Esto es excelente para hacer una copia de seguridad de tu trabajo.

3

Abre un archivo guardado

Haz clic en File → Open para cargar un programa previamente guardado de nuevo en el editor.

💡
Consejo Pro

Dale a tus archivos nombres descriptivos como blink_led.py o drive_square.py para que puedas encontrarlos fácilmente más tarde.
