💻 Welcome to the XRP Web Editor!

Now that your robot is built, it's time to make it do cool things! You'll write programs using a tool called the XRP Web Editor.

The XRP Web Editor — your programming home base!
🌐
Open the editor

Your teacher will give you the web address. Type it into your Chrome or Edge browser and press Enter. The editor works right in your web browser — no installation needed!

In this lesson you'll learn:

  • How to connect your robot to the editor
  • How to find your way around the editor
  • How to write your first Blockly program
  • How to run your code on the robot

🔌 Connecting Your Robot

Before you can program, you need to connect your robot to the computer with a USB cable.

Plug the USB cable into the robot and into your computer
1

Plug in the USB cable

Connect one end of the USB cable to the port on the robot's controller board, and the other end to your computer.

2

Turn on the robot

Flip the power switch on the controller board to the ON position. You should see a green LED light up.

3

Click "Connect" in the editor

In the web editor, click the Connect button at the top. A list of ports will appear. Choose the one that says something like "XRP" or "COM" and click Connect.

4

Look for the green indicator

When connected, the editor shows a green "Connected" indicator. You're ready to code!

🗺️ Tour of the Editor

Let's take a quick look at the main parts of the XRP Web Editor.

The main areas of the XRP Web Editor

Key Areas:

A

Toolbar (top)

Buttons to Connect, Run, Stop, and Save your program.

B

Block Toolbox (left)

Categories of blocks you can use. Click a category to see the blocks inside it.

C

Blockly Workspace (center)

The big area where you drag and connect blocks to write your program.

D

Python Code View (right side)

This shows you the Python code that your blocks generate. Very cool!

E

Terminal (bottom)

Shows messages from your robot — like if something went wrong, or if you used a print block.

🧱 What is Blockly?

Blockly is a way to write programs by snapping blocks together — just like puzzle pieces! Each block does one specific thing.

Blocks snap together to build a program — no typing needed!

✅ Blockly

Drag and drop colorful blocks. Hard to make typos. Great for beginners!

🐍 Python

Type the code yourself. More powerful, but you have to get every letter perfect.

🤔
Did you know?

When you use Blockly, the editor automatically converts your blocks into real Python code! Professional engineers use Python to program robots, satellites, and more.

📚 Block Categories

Blocks are organized into categories by color. Here are the most important ones for the NanoXRP:

Click any category to see the blocks inside
🚗

DriveTrain (rust orange)

Makes the robot drive forward, backward, and turn.

📡

Sensors (green)

Reads the distance sensor, line sensors, gyro, and more.

💡

Control Board (teal)

🎮

⚙️

Control (blue-gray)

✋ Your First Program

Let's make the robot blink its LED light! This is the classic "Hello World" of robotics.

Your first program: blink the LED on and off!

Build this program step by step:

1

Find "forever" loop

In the Control category, find the repeat forever block and drag it to the workspace.

2

Add LED On block

In Control Board, drag LED on into the forever loop.

3

Add a wait block

In Control, find wait [0.5] seconds and add it after LED on.

4

Add LED Off + another wait

Add LED off and another wait 0.5 seconds block.

🔁 repeat forever
💡 LED on

⏱ wait 0.5 seconds

💡 LED off

⏱ wait 0.5 seconds

▶️ Running Your Code

Once your blocks are in place, it's time to send the program to the robot and watch it run!

Click the green Run button to send your program to the robot
1

Make sure robot is connected

Check that the green "Connected" indicator is showing in the toolbar.

2

Click the Run button ▶️

The editor converts your blocks to Python and sends it to the robot. The Terminal will show what's happening.

3

Watch your robot!

The LED on the board should blink on and off every half second. Congratulations — you just ran your first robot program!

4

Stop the program

Click the red Stop button ⏹ to stop the program running. The LED will stop blinking.

⚠️
Something went wrong?

Check the Terminal area for red error messages. The most common issue is not being connected — try clicking Connect again.

💾 Saving Your Work

Saving your programs is important! Here's how to save and re-open your work.

Save your program so you can come back to it later
1

Save to Robot

Click File → Save to Robot. Your program is saved onto the robot itself as a file called main.py. This program runs automatically when the robot turns on!

2

Save to Computer

Click File → Save to save a copy to your computer. This is great for backing up your work.

3

Open a saved file

Click File → Open to load a previously saved program back into the editor.

💡
Pro Tip

Give your files descriptive names like blink_led.py or drive_square.py so you can find them easily later.
