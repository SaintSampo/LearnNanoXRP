¿Cómo funciona?

Un gamepad (como un controlador de Xbox o PlayStation) se conecta a tu computadora. El Editor Web XRP lee las posiciones del joystick y los botones presionados, luego envía esa información a tu robot a través de la conexión USB o Bluetooth.

Aprenderás:

  • Cómo conectar un gamepad a tu computadora
  • La distribución del gamepad — joysticks, botones y el D-pad
  • Cómo usar bloques de Gamepad en Blockly
  • Cómo programar arcade drive (un joystick controla velocidad y dirección)
  • Cómo usar botones para acciones adicionales

🎮 Conectando Tu Gamepad

Primero, conectemos el gamepad a tu computadora y hagamos que el Editor Web XRP lo reconozca.

Conecta el gamepad y busca el icono del controlador en el editor web
1

Enchufa el gamepad

Conecta tu gamepad a tu computadora usando su cable USB (o dongle receptor para gamepads inalámbricos). Windows debería instalar el controlador automáticamente.

2

Presiona cualquier botón en el gamepad

El navegador necesita que presiones un botón en el gamepad antes de reconocerlo. Presiona cualquier botón (A, B, etc.) una vez.

3

Busca el indicador del gamepad

En la barra de herramientas del Editor Web XRP, verás aparecer un icono de gamepad cuando se detecte el controlador. El icono se iluminará cuando muevas los joysticks.

4

Conecta tu robot también

Asegúrate de que tu robot esté conectado al editor a través de USB o Bluetooth — necesitas que tanto el gamepad como el robot estén conectados al mismo tiempo.

🕹️ Distribución del Gamepad

Aprendamos las diferentes partes del gamepad y cuáles son sus nombres en el sistema XRP.

Las partes principales de un controlador de gamepad estándar
Parte Nombre XRP Valores
Joystick izquierdo (izquierda/derecha) LEFT_X -1.0 (izquierda) a +1.0 (derecha)
Joystick izquierdo (arriba/abajo) LEFT_Y -1.0 (arriba) a +1.0 (abajo)
Joystick derecho (izquierda/derecha) RIGHT_X -1.0 (izquierda) a +1.0 (derecha)
Botones A, B, X, Y BTN_A, BTN_B, etc. 0 (soltado) o 1 (presionado)
Bumpers Izquierdo/Derecho BTN_L, BTN_R 0 o 1

🧱 Bloques de Gamepad en Blockly

La categoría Gamepad (naranja) tiene dos bloques para leer la entrada del gamepad.

Los dos bloques de Gamepad en Blockly
1

obtener valor gamepad [eje]

Devuelve la posición actual de un eje del joystick. Elige qué eje del menú desplegable (LEFT_X, LEFT_Y, RIGHT_X, RIGHT_Y, etc.).

🎮 obtener valor gamepad LEFT_Y

Devuelve un número de -1.0 a 1.0

2

¿botón presionado? [botón]

Devuelve VERDADERO si el botón seleccionado está presionado actualmente, FALSO si no.

🎮 ¿botón presionado? BTN_A

Devuelve Verdadero o Falso

🕹️ Arcade Drive

Arcade drive usa UN joystick para controlar tanto la velocidad como la dirección — como en la mayoría de los videojuegos de carreras. ¡XRP tiene un bloque arcade() integrado para esto!

🤔
Cómo funciona arcade drive
  • Empuja el joystick arriba → robot avanza hacia adelante
  • Empuja el joystick abajo → robot retrocede hacia atrás
  • Empuja el joystick izquierda → robot gira izquierda
  • Empuja el joystick derecha → robot gira derecha
  • ¡Empújalo en diagonal → avanza Y gira al mismo tiempo!
¡Un joystick = control completo del robot!
⚠️
¡El eje Y está invertido!

En la mayoría de los gamepads, empujar el joystick HACIA ARRIBA da un valor Y NEGATIVO (-1.0). Así que cuando lo programes, tendrás que invertir el valor Y: usa -(LEFT_Y) para que arriba = positivo (hacia adelante).

🧱 Construyendo el Programa Arcade Drive

Aquí está el programa completo arcade drive usando el joystick izquierdo. El robot continuamente lee el joystick y actualiza sus motores.

El programa completo arcade drive con gamepad
# Ejecutar para siempre, actualizando cada ciclo
🔁 repetir infinitamente
🚗 arcade drive
recto: - 🎮 obtener valor gamepad LEFT_Y
giro: 🎮 obtener valor gamepad LEFT_X
⏱ esperar 0.02
💡
¿Por qué invertir LEFT_Y?

La mayoría de los controladores reportan arriba como -1 y abajo como +1. Como queremos arriba = hacia adelante = positivo, invertimos el signo con un bloque de "negación" (el bloque de signo menos en la categoría Matemáticas).

🔘 Añadiendo Controles de Botones

¡Ahora añadamos botones para hacer cosas adicionales mientras conduces!

Ejemplo: Mantén presionado el botón A para activar claxon / parpadear LED

🔁 repetir infinitamente
🚗 arcade drive recto: -🎮 valorLEFT_Y giro: 🎮 valorLEFT_X
si 🎮 ¿botón presionado?BTN_A
💡 LED encendido
si no
💡 LED apagado
⏱ esperar 0.02

¡Ahora cuando mantengas presionado el Botón A, el LED se enciende! Puedes usar botones para activar cualquier acción: parpadear el LED, reproducir un sonido, detener el robot, etc.

🏆
Ideas para acciones de botones
  • Botón B: Claxon (parpadear LED rápidamente)
  • Botón X: Girar en el lugar
  • Botón Y: Parada de emergencia
  • Bumper izquierdo: Modo de velocidad lenta
  • Bumper derecho: Modo de velocidad turbo
